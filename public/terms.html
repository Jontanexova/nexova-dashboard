Condiciones del Servicio

Última actualización: 19 de abril de 2026

1. Aceptación

Al utilizar Nexova AutoPublish ("la Aplicación"), el operador acepta estas Condiciones del Servicio. Si no está de acuerdo, debe abstenerse de usar la Aplicación.

2. Naturaleza del servicio

Nexova AutoPublish es una herramienta de uso interno desarrollada por JMV Distribución / Nexova para automatizar la publicación de contenido multimedia en canales propios de redes sociales (YouTube, Facebook, Instagram, TikTok). No se ofrece como servicio público a terceros.

3. Uso aceptable

El operador se compromete a:

  • Usar la Aplicación únicamente con canales y cuentas de su propiedad o bajo su administración autorizada.
  • Cumplir los términos de servicio de cada plataforma integrada (YouTube, Meta, TikTok).
  • No publicar contenido que infrinja derechos de autor, incite violencia, o viole las políticas comunitarias de las plataformas.
  • No usar la Aplicación para prácticas de spam, manipulación artificial de engagement, o cualquier actividad prohibida por las plataformas.

4. Integraciones de terceros

La Aplicación se integra con APIs de terceros (Google YouTube Data API, Meta Graph API, TikTok Content Posting API). El uso de estas APIs está sujeto a los términos de cada proveedor:

  • YouTube API Services Terms of Service
  • Meta Platform Terms
  • TikTok Developer Terms

5. Limitación de responsabilidad

La Aplicación se proporciona "tal cual", sin garantías de disponibilidad continua. El operador es responsable de revisar el contenido publicado y del cumplimiento de las normas de cada plataforma.

6. Modificaciones

Estos términos pueden actualizarse. La fecha de "Última actualización" indica la versión vigente.
